使用Hugo搭建个人博客

在Linux上配置环境，使用Hugo + Github的组合搭建个人博客。

下载brew

下载Hugo要使用brew这个工具。

过程详见这篇博客：在 Linux 上安装和使用 Homebrew 包管理器

下载Hugo

使用如下命令下载Hugo：

brew install hugo

使用如下命令检验是否下载成功：

hugo version

若成功输出版本信息，说明下载成功：

hugo v0.115.4+extended linux/amd64 BuildDate=unknown

创建一个名为blog博客：

hugo new site blog

输出：

Congratulations! Your new Hugo site is created in /home/rex/blog.

Just a few more steps and you're ready to go:

1. Download a theme into the same-named folder.
   Choose a theme from https://themes.gohugo.io/ or
   create your own with the "hugo new theme <THEMENAME>" command.
2. Perhaps you want to add some content. You can add single files
   with "hugo new <SECTIONNAME>/<FILENAME>.<FORMAT>".
3. Start the built-in live server via "hugo server".

Visit https://gohugo.io/ for quickstart guide and full documentation.

下载并设置Hugo主题

Hugo没有默认主题，需要手动下载并设置一个。

可以前往Hugo官网下载：Hugo Themes

选择一个分栏风格的主题：hugo-theme-m10c

在blog目录下，运行如下命令，下载主题：

git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

启动博客服务器

使用如下命令，使用主题m10c，启动博客服务器:

hugo server -t m10c --buildDrafts

创建一篇名为self-blog的博客：

hugo new post/self-blog.md

将个人博客部署到远端服务器

在Github上创建一个名为Maxwell2020152049的仓库。

使用如下命令生成一个用于维护远端服务器网站的文件夹public:

hugo --theme=m10c --baseURL="https://Maxwell2020152049.github.io" --buildDrafts

进入public目录，使用以下命令将该目录推送至Maxwell2020152049仓库：

git init
git add .
git commit -m "create hugo blog"
git remote add origin git@github.com:Maxwell2020152049/Maxwell2020152049.github.io.git
git push --set-upstream origin master

然后就能够进入网站了。
